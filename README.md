# Tripleten web_project_around_es

# Descripción

Este proyecto es una aplicación web interactiva que permite a los usuarios visualizar, agregar y gestionar tarjetas con imágenes, así como editar su perfil.
La aplicación está conectada a un servidor mediante una API, lo que permite trabajar con datos reales de manera dinámica.
Fue desarrollada como parte de mi formación en TripleTen, aplicando conceptos de programación asíncrona y consumo de APIs.

# Objetivo

Construir una aplicación web dinámica que:

1.Se conecte a un servidor mediante una API
2.Permita gestionar información del usuario
3.Manipule datos en tiempo real desde el backend

# Tecnologías utilizadas

HTML5
CSS3
JavaScript (ES6+)
APIs REST
Manipulación del DOM
Git y GitHub

# Funcionalidades

1.Obtener información del usuario desde la API
2.Editar perfil (nombre, descripción y avatar)
3.Cargar tarjetas desde el servidor
4.Agregar nuevas tarjetas
5.Eliminar tarjetas propias
6.Dar y quitar "me gusta" a tarjetas
7.Confirmación antes de eliminar elementos
8.Visualización de imágenes en popup
9.Actualización dinámica de la interfaz
